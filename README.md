# Ubuntu ARM Server Lab                                                                                                                                                                            
                  
  A personal Linux lab running Ubuntu on an ARM VM. Used for hands-on practice                                                                                                                       
  alongside CompTIA Network+ studies.
                                                                                                                                                                                                     
  ## Environment  
  - OS: Ubuntu ARM Server                                                                                                                                                                            
  - Platform: VM (Mac host)
                                                                                                                                                                                                     
  ## Topics Covered
  - Linux CLI fundamentals                                                                                                                                                                           
  - Networking tools (nmap, tcpdump, netstat, ss)
  - Firewall configuration (ufw)                                                                                                                                                                     
  - SSH setup and key management
